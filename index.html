Android Studio 

Sketchware™ is an integrated development environment that uses Scratch to create and build android application directly on your android device

Simple Preparation

Sketchware replaces complicated development environment. Because we think programming experience is more valuable, you can get familiar with user-friendly development environment before you jump into complicated development environments.

Scratch

Scratch, a block-like programming language created by MIT, helps young people learn visually to think creatively, reason systematically, and work collaboratively — essential skills for life in the 21st century. (source - MIT)

Creativity

The ability to think creatively and critically is more important than understanding syntax. Uniquely express your creativity by programming applications using a lego-like block language, Scratch. Everyone has their own programming style.

Code Translation

The projects are automatically translated into necessary XML and Java source codes for those who want to continue their project on Android Studio. This can not only save a tremendous time when starting a simple boilerplate, but also give an insight to the programmer.
